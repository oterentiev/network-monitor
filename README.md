# Інформаційна система діагностики локальної мережі

## Як отримати EXE через GitHub

1. Створіть репозиторій GitHub.
2. Завантажте всі файли з цього архіву.
3. Виконайте Push у гілку main.
4. GitHub Actions автоматично створить EXE.

## Де знайти EXE

GitHub → Actions → Build Windows EXE → Artifacts

Там буде файл:

NetworkMonitor.exe
